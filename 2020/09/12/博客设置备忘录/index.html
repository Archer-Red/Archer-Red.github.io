发表于 2020-09-12 更新于 2020-09-13 分类于 学习笔记
静态博客搭建的一些操作

博客搭建流程根据参考网页

Hexo常用命令

参考网站

  • hexo init

    新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

  • hexo new “title” #默认目录post下添加博客
  • hexo new page “title” #source目录下添加title文件夹和title.md，可以用来菜单栏添加子目录
  • hexo new page –path about/me “About me”

    新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。第二个命令会创建一个 source/about/me.md 文件，同时 Front Matter 中的 title 为 “About me”

  • hexo g

    本地生成文件

  • hexo g -d

    本地生成后立刻部署到网上

  • hexo s

    启动服务器，访问网址为： http://localhost:4000/

  • hexo d

    部署网站

    网站采用next主题

    Hexo主题网站
    参考主题next配置教程
    更换主题： Hexo文件夹下的 themes目录，右键 Git Bash Here输入以下命令
    git clone 主题对应网址
    修改站点配置文件_config.yml,修改theme参数
    部署

菜单设置

1.编辑 主题配置文件，修改以下内容：设定菜单内容，对应的字段是 menu。 菜单内容的设置格式是：item name: link。其中 item name 是一个名称，这个名称并不直接显示在页面上，她将用于匹配图标以及翻译

若你的站点运行在子目录中，请将链接前缀的 / 去掉
2.设置菜单项的显示文本。在第一步中设置的菜单的名称并不直接用于界面上的展示。Hexo 在生成的时候将使用 这个名称查找对应的语言翻译，并提取显示文本。这些翻译文本放置在 NexT 主题目录下的 languages/{language}.yml （{language} 为你所使用的语言）。

3.设定菜单项的图标，对应的字段是 menu_icons。 此设定格式是 item name: icon name，其中 item name 与上一步所配置的菜单名字对应，icon name 是 Font Awesome 图标的 名字。而 enable 可用于控制是否显示图标，你可以设置成 false 来去掉图标。

4.添加新菜单需要
hexo new page "title"
source目录下添加title文件夹和index.md，可以用来菜单栏添加子目录
